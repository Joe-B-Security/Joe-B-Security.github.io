## About Me

I'm a security researcher focused on discovering and responsibly disclosing vulnerabilities in web applications and AI systems. My research interests include:

### Web Security
- Cross-Site Scripting (XSS) exploitation and defense
- Cross-Origin Resource Sharing (CORS) misconfigurations
- HTTP request smuggling and desync attacks
- Cookie security and SameSite bypasses
- Web Application Firewall (WAF) bypasses

### AI Security
- Prompt injection vulnerabilities and defenses
- Data exfiltration in Large Language Models
- Custom GPT security configurations
- AI system security assessments

### Security Tools & Automation
- Advanced reconnaissance techniques
- Google dorking at scale
- Vulnerability discovery automation
- Security testing methodologies

## Bug Bounty & Research

I actively participate in bug bounty programs and responsible disclosure. My findings have helped secure various applications and systems.
